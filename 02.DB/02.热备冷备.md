1. 备份的类型

   | Type                | Info                                          |
   | ------------------- | --------------------------------------------- |
   | 冷备（cold backup） | 需要关mysql服务，读写请求均不允许状态下进行； |
   | 温备（warm backup） | 服务在线，但仅支持读请求，不允许写请求；<br/> |
   | 热备（hot backup）  | 备份的同时，业务不受影响。                    |
